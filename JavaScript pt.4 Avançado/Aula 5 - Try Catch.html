Try Catch O Try Catch é utilizado para tratar erros que podem acontecer dentro do sistema. Esses erros podem ser por codificação do programador, entrada incorreta de dados pelo usuário ou até mesmo imprevistos. O try consegue recuperar erros que possam ocorrer no código fornecido em seu bloco. O catch por sua vez faz o tratamento dos erros que aconteceram. Além disso também temos um bloco chamado finally, que fornecem instruções do que deve ser executado após o try/catch. Tem 3 formas de declarações try : try...catch try...finally try...catch...finally
