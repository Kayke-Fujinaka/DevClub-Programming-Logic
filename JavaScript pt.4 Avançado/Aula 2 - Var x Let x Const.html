Var x Let x Const A principal diferença entre eles é o escopo. - As declarações de var tem escopo global ou de função, enquanto as declarações de let e de const têm escopo de bloco. - Variáveis de var podem ser atualizadas e declaradas novamente dentro de seu escopo. As variáveis de let podem ser atualizadas, mas não podem ser declaradas novamente. As variáveis de const não podem ser atualizadas nem declaradas novamente. - Todas elas passam por hoisting para o topo de seu escopo. Porém, enquanto variáveis com var são inicializadas com undefined, as variáveis com let e const não são inicializadas. - Enquanto var e let podem ser declaradas sem ser inicializadas, const precisa da inicialização durante a declaração.
